Lecture Notes

  • Lecture 1: Introduction, Classification, Validation and Testing
  • Lecture 2: Linear Classifiers
  • Lecture 3: Gradient Descent, Perceptrons, Hard-Margin SVM
  • Lecture 4: Soft-margin SVM, nonlinear features
  • Lecture 5: Machine learning abstractions, optimization problems
  • Lecture 6: Decision theory, Generative/discriminative models
  • Lecture 7: QDA, LDA, MLE of parameters
  • Lecture 8: Eigendecomposition, Spectral Theorem, anisotropic Gaussians
  • Lecture 9: More anisotropic Gaussians
  • Lecture 10: Regression overview, least-squares and logistic regression
  • Lecture 11: Newton's method and logistic regression; more least squares
  • Lecture 12: Statistical justifications for regression
  • Lecture 13: Ridge regression, Lasso, Subset selection
  • Lecture 14: Decision trees, entropy & info gain
  • Lecture 15: More decision trees, ensembles, random forests
  • Lecture 16: Kernels
  • Lecture 17: Neural networks, gradient descent, backprop
  • Lecture 18: Neuron biology, tricks and heuristics for NN training
  • Lecture 19: Heuristics for NN training, CNNs
  • Lecture 20: Unsupervised learning: PCA
  • Lecture 21: SVD, k-means and other clustering techniques
  • Lecture 22: Spectral graph clustering
  • Lecture 23: More spectral graph clustering, random projection, high-dim spaces
  • Lecture 24: AdaBoost, nearest neighbors
  • Lecture 25: k-NN algorirthm, k-d trees, Conclusion
